# Job Profile Repository

This repository is the single source of truth for my professional career materials.

## Goals

- Maintain a master resume in Markdown.
- Track accomplishments continuously.
- Generate tailored resumes for different roles.
- Keep all career assets version controlled.

## Repository Structure

```text
.
├── README.md
├── CHANGELOG.md
├── accomplishments/
├── cover-letters/
├── linkedin/
├── portfolio/
└── resume/
```

## Working Approach

- Update `resume/master-resume.md` first.
- Derive shorter or role-specific resumes from the master resume.
- Log meaningful wins in `accomplishments/`.
- Capture major milestones in `CHANGELOG.md`.

## Suggested Workflow

1. Build the canonical resume in `resume/master-resume.md`.
2. Keep accomplishments current throughout the year.
3. Refresh LinkedIn copy from the same source material.
4. Tailor role-specific resumes and cover letters as needed.
